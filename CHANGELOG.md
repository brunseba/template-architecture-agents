# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-02-14

### Added
- Initial tools catalog with comprehensive descriptions for 20 tools
- Organized tools into 12 logical categories:
  - Diagram and Visualization Tools (Mermaid, Draw.io, Draw.io MCP)
  - Architecture Modeling Tools (Archi, Open Group Standards)
  - Requirements and Product Definition (GitHub Spec Kit)
  - Methodology and Frameworks (BMAD Method, Azure Well-Architected)
  - Model Context Protocol Tools (AWS Labs MCP, CucumberStudio MCP)
  - Testing and Quality Assurance (Gherkin)
  - Research and Discovery (Semantic Scholar)
  - Development Tools and Extensions (VS Code Extension Samples)
  - Kubernetes and Container Orchestration (Operator Pattern, Kubernetes API)
  - Data Standards and Schemas (JSON Schema)
  - Data Management and Governance (DataHub, OpenMetadata)
  - Versioning and Release Management (SemVer, Conventional Commits)
- Usage and integration guidelines
- Recommended tool combinations for common workflows
- Version tracking system (VERSION file and metadata in documentation)
